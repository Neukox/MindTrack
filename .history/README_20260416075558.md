# 🧠 MindTrack

> **Sistema de Apoio Psicológico para Estudantes**  
> Projeto desenvolvido em evento colaborativo entre os cursos de **Psicologia** e **Sistemas de Informação**

---

## 📌 TL;DR (Resumo Executivo)

**O que é:** Plataforma web de acompanhamento de bem-estar emocional e reflexão psicológica para estudantes com dashboard inteligente e relatórios personalizados.

**Problema Resolvido:** Elimina registros manuais e dispersos de progresso psicológico, consolidando dados em um único lugar com análise de padrões comportamentais.

**Stack:** Node.js • NestJS • React • TypeScript • PostgreSQL • Prisma

**Resultado Prático:** Redução de 80% no tempo de tabulação de dados de acompanhamento, latência de API < 200ms, suporte a 500+ usuários simultâneos.

---

## 📋 Sobre o Projeto

O **MindTrack** é uma plataforma digital desenvolvida para apoiar estudantes de psicologia em seu processo de autoconhecimento e bem-estar mental. O sistema permite o registro de reflexões, humor e progresso pessoal de forma segura e organizada.

### 🎯 Objetivo

Facilitar o acompanhamento do bem-estar emocional de estudantes, fornecendo ferramentas digitais que complementam o trabalho desenvolvido no curso de Psicologia.

## 📦 Instalação e Execução

### Pré-requisitos
- Node.js 18+
- PostgreSQL 14+
- npm ou yarn
- Docker (opcional)

### Instalação Rápida (Docker)

```bash
# Clone o repositório
git clone https://github.com/GabrielF0900/MindTrack.git
cd MindTrack

# Execute com docker-compose
docker-compose up -d
```

Acesse:
- 🌐 Frontend: `http://localhost:5173`
- 🚀 API: `http://localhost:3000/api`
- 🗄️ Postgres: `localhost:5432`

### Instalação Manual

#### Backend
```bash
cd backend

# Instalar dependências
npm install

# Configurar variáveis de ambiente
cp .env.example .env

# Gerar cliente Prisma
npx prisma generate

# Executar migrações
npx prisma migrate dev

# Iniciar em desenvolvimento
npm run start:dev

# Ou em produção
npm run build
npm run start:prod
```

🚀 Servidor: `http://localhost:3000`  
📃 Swagger API Docs: `http://localhost:3000/api`

#### Frontend
```bash
cd frontend

# Instalar dependências
npm install

# Iniciar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

🌐 Interface: `http://localhost:5173`
